Modal

×
How to close:
  1. Click × at top right corner
  2. Click on backdrop
  3. Press close modal button below
  4. Press ESC



HTML: 

  <button class="show-modal">show modal</button>

  <div class="modal"></div>
    <div class="m-backdrop"></div>
    <div class="m-content">
      <span class="modal-close">×</span>
      <!-- modal content goes here... -->
      <button class="modal-button">close modal</button>
    </div>
  </div>


CSS: 

  .modal { display: none; }

  .m-backdrop {
    background-color: rgba(0, 0, 0, .5);
    position: fixed;
    top: 0;
    right: 0;
    bottom: 0;
    left: 0;
  }

  .m-content {
    background-color: #f0f0f0;
    position: fixed;
    left: 50%;
    top: 50%;
    transform: translate(-50%, -50%);
    width: 500px;
    height: 300px;
    border: 1px solid white;
    border-radius: 4px;
  }

  .modal-close {
    position: absolute;
    top: 0px;
    right: 7px;
    font-size: 27px;
    font-weight: bold;
    cursor: pointer;
    color: lightgray;
  }

  .modal-close:hover { color: gray; }


JS: 

  // open modal
  $(".show-modal").on("click", function () {
    $(".modal").css("display", "block");
  });

  // close modal
  $(".modal-close").on("click", function () {
    $(".modal").css("display", "none");
  });

  $(".modal-button").on("click", function () {
    $(".modal").css("display", "none");
  });

  $(".modal").on("click", ".m-backdrop", function () {
    $(".modal").css("display", "none");
  });

  $(document).on("keyup", function (event) {
    var code = event.keyCode || event.which;
    if (code == 27) { $(".modal").css("display", "none"); }
  });



3D rotation

x y z



HTML: 

    <div id="logo">
      <div id="cube">
        <img src="your-image.png" id="front">
        <img src="your-image.png" id="back">
      </div>
    </div>


CSS: 

    #logo {
      -webkit-perspective: 800px;
      perspective: 400px;
    }

    #cube {
      height: 120;
      width: 120;
      -webkit-transform-style: preserve-3d;
      -webkit-transform: rotateX(0) rotateY(0) rotateZ(0);
      transform-style: preserve-3d;
      transform: rotateX(0) rotateY(0) rotateZ(0);
      -webkit-animation: animate-y 5s infinite linear;
      animation: animate-y 5s infinite linear;
    }

    @-webkit-keyframes animate-y {
      0%   { -webkit-transform: rotateY(0); }
      100% { -webkit-transform: rotateY(360deg); }
    }

    @keyframes animate-y {
      0%   { transform: rotateY(0); }
      100% { transform: rotateY(360deg); }
    }

    #front {
      position: absolute;
      height: 120;
      width: 120;
      -webkit-backface-visibility: visible;
      -webkit-transform: translateZ(100px);
      backface-visibility: visible;
      transform: translateZ(50px);
    }

    #back {
      position: absolute;
      height: 120;
      width: 120;
      -webkit-backface-visibility: visible;
      -webkit-transform: rotateY(180deg) translateZ(100px);
      backface-visibility: visible;
      transform: rotateY(180deg) translateZ(50px);
    }
